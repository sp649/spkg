# spkg - SP649's Package Manager (WIP)
A small simple package manager written in Go using sml markdown files.
### Usage:
```bash
spkg install <package|file.sml>
spkg remove <package|file.sml>
spkg update <package|file.sml|@all>
spkg search <package>
spkg list
```
